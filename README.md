# Student Performance Analysis

## Overview
This project analyzes the performance of students based on various attributes, such as test preparation, socio-economic status, gender, and parental education levels. The aim is to uncover insights that can inform educational strategies and support systems.

## Data Description
The dataset used for this analysis is contained in a CSV file named `StudentsPerformance.csv`. It includes the following columns:

- `gender`: Gender of the student.
- `race/ethnicity`: Group classification.
- `parental level of education`: Educational attainment of the parents.
- `lunch`: Type of lunch received (standard or free/reduced).
- `test preparation course`: Course completion status.
- `math score`: Score in mathematics.
- `reading score`: Score in reading.
- `writing score`: Score in writing.
